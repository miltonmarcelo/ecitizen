# eCitizen

Web based citizen issue reporting and case management platform.

## Tech Stack
- React with Vite
- Node.js with Express
- Firebase Authentication
- Prisma ORM
- SQLite

## Phase 1 Goals
- Repo ready
- React app running
- Express app running
- Firebase project created
- Prisma connected to SQLite
- Initial database schema drafted
- Route and page list defined
- Login flow decided
